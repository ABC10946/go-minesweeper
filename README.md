
# go-minesweeper

小さな CLI ベースのマインスイーパー実装（Go）。学習用途およびシンプルなゲームロジックの実演を目的としています。

**特徴**
- コンソール上で遊べるシンプルなマインスイーパー
- ゲームロジックは `minesweeperlogic` パッケージに分離
- コマンドライン操作を `cmd` 配下で管理

**必要環境**
- Go 1.18+ がインストールされていること（`GOPATH` は不要です）

**ビルドと実行**

開発中にすぐ動かす（ビルド不要）:

```bash
go run main.go
```

バイナリをビルドして実行する:

```bash
go build -o minesweeper .
./minesweeper
```

（シェルが `zsh` でも同じコマンドで問題ありません）

**ディレクトリ構成（主要ファイル）**

- `main.go` - エントリポイント
- `cmd/root.go` - CLI のエントリ／コマンド定義
- `game/game.go` - ゲームフロー（描画や入力処理）
- `minesweeperlogic/minesweeperlogic.go` - マインスイーパーのコアロジック

実装の詳細は各ファイルを参照してください。

**使い方（基本）**

1. コマンドを実行するとゲームが開始します。
2. 画面の指示に従ってマスを開く（またはフラグを立てる）操作を行ってください。

（UI の具体的な操作方法やキー割り当ては実装に依存します。必要であれば README を更新します。）

**開発/テスト**

- 依存管理は Go Modules を使用しています。外部依存が増えた場合は `go mod tidy` を実行してください。

```bash
go mod tidy
```

単体テストが含まれている場合は次のコマンドで実行できます（現状テストが無ければ何も実行されません）:

```bash
go test ./...
```
